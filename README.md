# Todo List CLI

Een command-line interface (CLI) applicatie om je todo-lijst bij te houden met persistente opslag in JSON.

---

## **Features**

- Todos toevoegen
- Todos bekijken
- Todos markeren als voltooid
- Persistente opslag in `todos.json`
- Eenvoudige en overzichtelijke command-line interface

---

## **Installatie**

1. Clone de repository:

```bash
git clone git@github.com:username/todo-app.git
cd todo-app
```
